# PRD - BioSync

## 프로젝트 개요
BioSync는 모바일 앱에서 수집된 생체신호 데이터를 저장, 분석, 조회하는 플랫폼이다.
사용자는 건강 데이터를 확인할 수 있고, 관리자는 관리자 웹에서 사용자 및 시스템 상태를 모니터링할 수 있다.

## 프로젝트 목표
- Spring Boot 기반 모바일 백엔드 설계
- 생체신호 데이터 수집 및 분석 시스템 구현
- 사용자 / 관리자 시스템 분리
- 확장 가능한 데이터 구조 설계

## 주요 사용자

### USER
모바일 앱 사용자
- 회원가입 / 로그인
- 디바이스 등록
- 생체신호 업로드
- 일별 건강 데이터 조회
- 이상징후 확인

### ADMIN
관리자
- 사용자 목록 조회
- 특정 사용자 데이터 조회
- 이상징후 조회
- 시스템 수집 현황 확인

## 핵심 기능

### 인증
- 회원가입
- 로그인
- JWT 인증

### 디바이스 관리
- 디바이스 등록
- 디바이스 목록 조회

### 생체신호 업로드
지원 데이터
- 심박수
- 걸음 수
- 수면 시간
- 스트레스 지수

업로드 방식
Batch Upload

### 데이터 분석
- 평균 심박수
- 최대 심박수
- 최소 심박수
- 총 걸음 수
- 총 수면 시간

### 이상징후 탐지
예시
- 심박수 120 초과
- 하루 걸음 수 급감

### 관리자 기능
- 사용자 조회
- 이상징후 조회
- 데이터 수집 현황 조회

## MVP 제외 범위
