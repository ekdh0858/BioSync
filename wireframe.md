# Wireframe

이 문서는 BioSync의 주요 화면을 텍스트 와이어프레임으로 정리한 것이다. 사용자 앱은 모바일 우선, 관리자 화면은 데스크톱 우선으로 설계한다.

## 1. 사용자 앱 홈

목적: 오늘의 건강 상태와 최근 알림을 빠르게 확인한다.

```text
+--------------------------------------------------+
| BioSync                              알림  설정 |
+--------------------------------------------------+
| 안녕하세요, Taehun                               |
| 오늘 상태: 안정적                                |
+--------------------------------------------------+
| 오늘의 요약                                       |
| 심박수 평균   72 bpm                             |
| 걸음 수      8,420                               |
| 수면 시간    6h 45m                              |
+--------------------------------------------------+
| 최근 알림                                         |
| - 심박수 상승 감지: 08:40                        |
| - 데이터 업로드 완료: 07:00                      |
+--------------------------------------------------+
| 하단 탭: 홈 | 기록 | 업로드 | 알림 | 마이        |
+--------------------------------------------------+
```

## 2. 사용자 기록 화면

목적: 날짜별 Summary와 원본 데이터 추이를 확인한다.

```text
+--------------------------------------------------+
| 기록                                              |
+--------------------------------------------------+
| 기간 선택: [오늘] [7일] [30일]                   |
+--------------------------------------------------+
| 일별 Summary 카드                                 |
| 2026-03-16                                       |
| 평균 심박수   72 bpm                             |
| 최대 심박수   118 bpm                            |
| 걸음 수       8,420                              |
| 수면 시간     6h 45m                             |
+--------------------------------------------------+
| 차트 영역                                         |
| 심박수 추이                                       |
| 걸음 수 추이                                      |
+--------------------------------------------------+
```

## 3. 사용자 업로드 화면

목적: 디바이스 연결 상태와 최근 업로드 결과를 확인한다.

```text
+--------------------------------------------------+
| 업로드                                            |
+--------------------------------------------------+
| 연결된 디바이스                                   |
| Galaxy Watch 6                                   |
| 상태: 연결됨                                      |
+--------------------------------------------------+
| 최근 업로드                                       |
| 2026-03-16 07:00 성공                             |
| 2026-03-15 22:00 성공                             |
| 2026-03-15 07:00 실패                             |
+--------------------------------------------------+
| [수동 업로드]   [오류 로그 보기]                  |
+--------------------------------------------------+
```

## 4. 사용자 알림 화면

목적: Alert를 심각도 순으로 확인한다.

```text
+--------------------------------------------------+
| 알림                                              |
+--------------------------------------------------+
| 필터: [전체] [주의] [위험]                       |
+--------------------------------------------------+
| 위험                                              |
| 심박수 120 초과                                   |
| 발생 시각: 2026-03-16 08:40                      |
+--------------------------------------------------+
| 주의                                              |
| 걸음 수가 최근 3일 평균보다 낮습니다              |
| 발생 일자: 2026-03-16                            |
+--------------------------------------------------+
```

## 5. 관리자 대시보드

목적: 사용자 수, 업로드 현황, 이상징후 발생 상황을 한 화면에서 확인한다.

```text
+--------------------------------------------------------------------------------+
| BioSync Admin                         검색        관리자명   로그아웃           |
+--------------------------------------------------------------------------------+
| Sidebar: Dashboard | Users | Signals | Alerts | Settings                      |
+--------------------------------------------------------------------------------+
| KPI Cards                                                                       |
| 전체 사용자 1,240 | 오늘 업로드 성공 1,102 | 위험 Alert 18 | 수집 실패 24     |
+--------------------------------------------------------------------------------+
| 좌측: 업로드 성공률 차트              우측: Alert 심각도 분포                  |
+--------------------------------------------------------------------------------+
| 최근 위험 Alert 목록                                                          |
| userId | 유형        | 발생 시간           | 상태                              |
| 1023   | HIGH_HEART | 2026-03-16 08:40    | 신규                              |
+--------------------------------------------------------------------------------+
```

## 6. 관리자 사용자 상세

목적: 특정 사용자 상태와 최근 수집 데이터를 조사한다.

```text
+--------------------------------------------------------------------------------+
| 사용자 상세: userId 1023                                                      |
+--------------------------------------------------------------------------------+
| 기본 정보     | 최근 디바이스 | 최근 로그인 | 최근 업로드 상태                 |
+--------------------------------------------------------------------------------+
| 최근 7일 Summary 차트                                                        |
+--------------------------------------------------------------------------------+
| 최근 Alert 목록                                                               |
| 최근 업로드 이력                                                              |
+--------------------------------------------------------------------------------+
```

## 7. 화면 설계 메모

- 사용자 앱은 핵심 수치 3개를 첫 화면 상단에 고정한다.
- Alert는 색상과 텍스트 라벨을 함께 사용한다.
- 관리자 화면은 표와 필터를 우선 배치한다.
- 모든 날짜와 시간은 절대값으로 표시한다.
